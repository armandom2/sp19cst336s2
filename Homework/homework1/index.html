A programming language is a set of commands,
instructions, and other syntax used to create a
software program. Programmers write their code
in what is known as a "high-level languages."
The code is then compiled through a "low-level
language,"which is recognized directly by the
computer hardware and is assembled to create an
output.

"High-level languages" allows programmers to
write source code in a natural fashion, using
key words and proper syntax. Examples of key
works are: function, while, if, and
else.Examples of Symbols are: ,
==, and != . Most
high-level are produced the same allowing
programmers to understand code in different
language and pick it up faster.

Low-level languages include assembly and machine
languages. An assembly language is built through
list of instructions, which are more complex than
a High-level language. An assembler is used to
translate the assembly code into machine code.
The machine code, or machine language, produces a
series of binary codes that are read by a computer's
CPU which give the High-level language functionality.
