History

Java was first developed in 1991, by James Glosing from Son
Microsystem and his team, known as the Green Team.The
intitial thought of java was desinged for interactive
television, which are controlled by multiple computer
processors.

Needing to get the new languages accessible by computer
processors, the team thought the language would be better off
being used use throughout web browsers. This lead to the
conncetion of Java to the Internet.

Initialy the name of the programming language was named,
Oak. As a result of trade-mark issues, Glosing's team had to
change the name. After serveral time being spent picking a
name, the team came up with Java while getting coffee.

In 1995, Netscape Incorporated released its latest version
of the Netscape browser which was capable of running Java
programs. Now Java is used internet programming, mobile
devices,games, e-business solutions, etc.
