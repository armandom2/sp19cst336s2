History

In the 1980s Van Rossum, the creator of python, was working
at the CWI(Centrum voor Wiskunde en Informatica)
implementing a program know as ABC.Soon after Rossum
was put to another task, working on a new distributing
operation system called AMOEBA. After being so used to
the ABCs syntax,Rossum did not like the AMOEBA layed
out, but enjoyed the system calls. To make things easier,
Rossum started developing a new scripting language which
would incorperate both ABC syntax and AMOEBA system
calls.

As Van Rossum deleloped the new script throughout the late
1980's, he released the first final version in 1991.The
initial release had module system of Modula-3.

Van Rossum decided to name this new scripting language
‘Python’. throughout the early release people thought the
program was named after the snake 'Python.' Rossum,
loving a popular BBC comedy tv show called Monty
Python’s Fly Circus Rossum, decided to name this new
scripting language after the tv show being 'Python.'
